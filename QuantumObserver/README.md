# 観測者は、もういない

量子力学の観測問題を題材にした、10〜15分想定の短編ホラーノベルです。

## 起動方法

1. TyranoBuilderで `QuantumObserver` フォルダをプロジェクトとして開きます。
2. `data/scenario/first.ks` を選び、プレビューを実行します。

ブラウザで確認する場合は、このフォルダでローカルHTTPサーバーを起動し、`index.html` を開いてください。

## 内容

- 選択肢：2回
- エンディング：3種類（収束／重ね合わせ／観測者）
- 専用背景・イベントCG：3枚
- 強い恐怖表現、流血、死体描写、画面揺れを含みます。

## 主な編集箇所

- シナリオ：`data/scenario/first.ks`
- 背景・イベントCG：`data/bgimage/`
- ゲーム設定：`data/system/Config.tjs`
